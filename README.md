## Getting Started

### Prerequisites
- NodeJs
- Pnpm

<hr/>

## Installation
Clone the repository and navigate to the project directory:
```
bash
git clone https://github.com/alvarezjuanign/landingLaArboleda.git
cd landingLaArboleda
```

Install the dependencies:
```
bash
pnpm install
```

Run the project:
```
bash
pnpm run dev
```

Open your browser and navigate to:
```
http://localhost:4321/
```
